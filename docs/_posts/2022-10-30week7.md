其实没啥好讲的内容，这周的作业很简单，唯一的问题是：

第二题：获取最高位的时候，很多同学写出如下的代码：

```cpp
...
// here we assume n >= 0
int highest;
while (n != 0) {
    highest = n;
    n /= 10;
}
...
```

当 `n == 0` 时，`highest` 的值不确定（未初始化），需要注意规范问题。

当然你也可以直接递归：

```cpp
int highest(int n) {
  if (n < 10 && n > -10) {
    return n;
  } else {
    return highest(n / 10);
  }
}
```
